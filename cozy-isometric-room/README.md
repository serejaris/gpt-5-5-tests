# 3D Agent Project Studio

Interactive Three.js demo prototype for managing project agents in a 3D office.

## Run

```bash
npm install
npm run dev
```

## Demo Interactions

- Hover and click low-poly agents in the office.
- Use the right panel to assign, pause, escalate, or complete demo work.
- Click pipeline zones to inspect local task queues.
- Use the dock for overview, simulation, focus, and reset.

There is no real agent/backend connection. All agent, task, and status data is local demo state.
