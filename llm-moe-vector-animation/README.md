# LLM and MoE vector animation

Standalone SVG/HTML animation that explains:

- how an LLM turns text into tokens and vectors;
- how attention mixes context;
- where the dense feed-forward block sits in a standard Transformer;
- how Mixture of Experts replaces that dense FFN path with a router and selected experts.

Open `index.html` directly in a browser or serve the folder with any static server.
